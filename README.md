# data_extraction

Enterprise level generic data extraction pipeline using Open Source Tools (PyMuPDF, pdfplumber, Tesseract OCR, Ollama Vision LLM).

## Features
- Extracts raw text, tables, and images from PDFs.
- Automatic OCR fallback for scanned images using Tesseract.
- Vision LLM (LLaVA via Ollama) integration for layout and table reconstruction.
